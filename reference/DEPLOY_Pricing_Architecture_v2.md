## PRICING PHILOSOPHY

DEPLOY pricing follows the same gated wave cascade logic as COMMAND with one critical distinction:

**COMMAND** sells operational infrastructure — a better way to run.
**DEPLOY** sells outcomes — campaigns, objectives, wins.

Outcomes command a higher price floor. The ROI argument for DEPLOY is stronger than COMMAND, not weaker. A financial operation returning $500K through a DEPLOY campaign makes the license fee a rounding error. Pricing reflects this.

## SEAT TRANSFERABILITY

All seats in both COMMAND and DEPLOY are transferable — between people and between products.

- Seats may be transferred to another party at any time
- Seats may be transferred across products — a COMMAND seat holder and a DEPLOY seat holder can work that out between themselves
- A buyer who wants a seat not currently available through the wave cascade can acquire one from an existing holder looking to transfer
- Ello Cello LLC receives royalty on transfer (rate per agreement)
- Seat appreciates in value as subsequent waves fill at higher prices

**Effect:** Seats are assets, not just licenses. The market handles liquidity naturally.

## BUNDLE PRICING

COMMAND licensees receive matched bundle pricing on DEPLOY — locked at their wave tier at time of COMMAND purchase.

| Scenario | Pricing |
|---|---|
| COMMAND only | COMMAND wave rate |
| DEPLOY only | DEPLOY standalone rate (higher floor) |
| COMMAND + DEPLOY together | Matched bundle — locked at buyer's wave tier |
| COMMAND first, DEPLOY later | DEPLOY at current wave rate (bundle window closed) |
| DEPLOY first, COMMAND later | COMMAND at current wave rate (bundle window closed) |

**Key principle:** The bundle rewards simultaneous commitment. Sequential purchases pay sequential prices. Bundle window closes when corresponding wave fills — permanently.

## DISTRIBUTION — PRIVATE

Not a seat. Not a wave product. Never advertised publicly.

Exists only in the buyer package for existing licensees. Invitation only — you are approached or you earn the conversation through demonstrated ecosystem commitment.

**What it is:**
A standing distribution agreement. Partner carries DEPLOY and COMMAND into markets Ello Cello LLC is not actively covering. MO§ES™ brand stays on the product. Royalty flows back on every license or campaign generated through their channel.

**What it is not:**
- Not white-label — brand does not change
- Not a seat — not capped, multiple partners possible
- Not public — no advertising, no listing

**Relationship to white-label:**
Distribution agreements are scoped to markets white-label does not cover. White-label terms are concluded before distribution agreements activate in any overlapping domain. No white-label buyer is surprised by distribution activity in their territory.

**Terms:**
- Invitation only
- Separate agreement negotiated case by case
- Royalty-based — rate TBD per partner
- Uncapped — no structural limit on distribution network size
- Reversion clause — lapses on inactivity
- MO§ES™ core excluded in all cases

## WHY PRICES INCREASE ACROSS WAVES

Same logic as COMMAND. Each wave opens only after the previous fills — proving demand with confirmed transactions. Early movers are rewarded with the lowest price and the most appreciable seat value.

DEPLOY adds one dynamic: **COMMAND licensees who bundle at Wave 1a lock the lowest combined price point across both products.** That window closes permanently when Wave 1a fills.
