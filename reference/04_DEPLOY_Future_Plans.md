## NEAR-TERM EXPANSION

### Formation Editor
Users build custom formations beyond the library. Define slot count, slot roles, communication flow, posture defaults. Save as named formation. Share or keep private.

### AI-Recommended Formation
Based on objective category, available agents, and posture selection — system suggests optimal formation. User confirms or overrides. Reduces decision overhead for new users.

### Responsive Layout Evolution
V1 is three-tab for accessibility and mobile compatibility. As the product matures, desktop evolves toward the full three-column cockpit:
- Left column: Inventory / Roster
- Center column: Field / Formation / Active Campaign
- Right column: Campaign List / Live Dashboard

Three-tab remains for mobile. Desktop gets the full cockpit.

### Individual Agent Conversation Threads
Per-agent conversation panel accessible from the live campaign view. Click any active agent in the formation to open their thread — see what they're doing, intervene if needed, review output.

---

## MID-TERM EXPANSION

### Cross-Domain Hybrid Formations
Combine metaphor domains. Military flanking structure with sports rotation logic. Gaming role composition with business posture. Formation designer allows mixing.

### Live Formation Shifts
Mid-campaign posture and formation changes based on real-time conditions. Operator calls an audible. Formation shifts. Agents adjust. Campaign continues without restart.

### Rotation Trigger Automation
Define conditions that automatically shift formation or posture — time elapsed, output threshold reached, adversarial signal detected, objective phase completed. System executes the shift without operator intervention.

### Campaign Templates
Pre-built campaign configurations for common use cases. Research blitz. Content sprint. Financial rebalance. Code audit. User selects template, adjusts to their roster, launches. Reduces setup time to near zero for known mission types.

## LONG-TERM EXPANSION

### Squad Marketplace
*(Aligned with SigEconomy™ longer term)*
Proven squad configurations shared or licensed between users. A financial trader's defensive rebalancing squad. A growth team's content offense squad. Lineage-tracked. Attribution maintained. Royalty on downstream use.

### Campaign Analytics
Which formations performed. Drift rates per agent. Objective completion scoring. Win condition tracking across campaign history. Performance data feeds back into formation recommendations.

### DEPLOY + CIVITAS Integration
DEPLOY campaigns generating SigRank™ data. Campaign performance contributing to operator signal scores. Long-game campaigns visible as civic events in SigTV™. The full MO§ES™ ecosystem compounding.

### Multi-Operator Campaigns
Multiple human operators running coordinated formations simultaneously toward a shared objective. War room mode — multiple commanders, one campaign.

### Financial Operations Deep Build
Agent wallet integration. On-chain execution support. Portfolio formation templates — defensive rebalancing, offensive position building, scout mode market survey. Governance layer specifically tuned for financial risk parameters.
